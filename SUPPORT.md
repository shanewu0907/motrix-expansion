## Suggested Issue Topics

Please include the following when reporting a problem:

- Browser and version
- Extension version
- Operating system
- Whether Motrix is running
- RPC host, port, and path
- Whether an RPC secret is required
- Whether cookie forwarding or referer forwarding is enabled
- Steps to reproduce the issue

## Common Troubleshooting

### The extension shows disconnected or auth failed

- Confirm Motrix or aria2 is running
- Confirm the RPC host, port, path, and secret are correct
- Test the connection from the Settings page

### A download stays in the browser instead of being sent to Motrix

- Confirm Motrix integration is enabled in the popup
- Confirm download takeover is enabled in Settings
- Check recent logs in the Settings page

### The extension cannot wake Motrix

- Confirm the custom URI or desktop app association is installed correctly
- Try launching Motrix manually and testing again
